The syntax of the TShark command typically follows this format: tshark [options] [capture filter] [display filter] Some commonly used options and filters include: ●-i: Specify the interface for packet capture. ●-f: Apply a capture filter to capture specific packets. ●-Y: Apply a display filter to filter displayed packets. ●-w: Write captured packets to a file. ●-R: Read packets from a capture file for analysis. ●-T: Specify the output format (e.g., text, JSON, PDML).
