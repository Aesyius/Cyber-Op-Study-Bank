Some common TCP port numbers and their associated services include: ●Port 80: HTTP (Hypertext Transfer Protocol) for web browsing. ●Port 443: HTTPS (Hypertext Transfer Protocol Secure) for secure web browsing. ●Port 25: SMTP (Simple Mail Transfer Protocol) for email transmission. ●Port 21: FTP (File Transfer Protocol) for file transfers. ●Port 22: SSH (Secure Shell) for secure remote access. ●Port 23: Telnet for remote login. ●Port 110: POP3 (Post Office Protocol version 3) for email retrieval. ●Port 143: IMAP (Internet Message Access Protocol) for email access. ●Port 3306: MySQL database server. ●Port 3389: RDP (Remote Desktop Protocol) for remote desktop access.
