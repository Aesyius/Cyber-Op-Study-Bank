Common FTP commands for file transfer and directory navigation include: ●GET: Download a file from the server to the client. ●PUT: Upload a file from the client to the server. ●LS: List files and directories on the server. ●CD: Change the current directory on the server. ●MKDIR: Create a new directory on the server. ●DELETE: Delete a file on the server. ●ASCII/BINARY: Set the data transfer mode (ASCII for text files, binary for non-text files).
